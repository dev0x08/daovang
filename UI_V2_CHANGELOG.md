# UI Pass v2 — Icon & Card Redesign

## Đã thay đổi
- Hướng dẫn: biểu tượng vai trò, tổng quan và kỹ năng được thống nhất thành huy hiệu dark-fantasy.
- Lá Mảnh Đường: thay icon Route lặp lại bằng 6 sơ đồ SVG riêng, thể hiện đúng cạnh mở của từng mảnh.
- Shop: card vật phẩm mới, thumbnail lớn, màu theo từng skin, bộ lọc Tất cả/Bàn cờ/Quân cờ/Hồ sơ.
- Nhiệm vụ & Achievement: icon riêng theo mục tiêu, progress bar có phần trăm, badge Coin/EXP và trạng thái rõ ràng.
- Responsive cho desktop, tablet và mobile.

## Kiểm tra
- `npm install`
- `npm run build` thành công.
